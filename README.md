## バッジの付け方
1. まず、以下のURLの検索欄でgithubと入力し、必要なものをクリックする。  
https://shields.io/

2. 次に、必要なもの（repoやuser等）を記入し、urlを生成し、README.mdに貼り付ける。
3. README.mdでは以下のように記述する。
```
[![名前](コピーしたurl)](リンク飛びしたい部分)
```
